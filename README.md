# regview
Offline-viewer for registry files

![](doc/regview_sample.png)

# Usage

```shell
regview 0.1.0

Offline-viewer for registry files

USAGE:
    regview <REG_FILE>

ARGS:
    <REG_FILE>    path to registry hive file

OPTIONS:
    -h, --help       Print help information
    -V, --version    Print version information
```
