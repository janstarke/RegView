# regview
Offline-viewer for registry files

![](doc/regview_sample.png)

# Usage

```shell
regview 0.2.1

Offline-viewer for registry files

USAGE:
    regview [FLAGS] <REG_FILE>

FLAGS:
    -h, --help               Prints help information
    -V, --version            Prints version information

ARGS:
    <REG_FILE>    path to registry hive file
```
